This chrome extension darkens the background color of Gmail.

# Usage
English
```
More tools
-> Extensions
-> Enable Developer mode
-> "Load unpacked"
-> Specify /blackGmail directory
```
Japanese
```
その他のツール  
-> 拡張機能  
-> デベロッパーモードをON  
-> "パッケージ化されていない拡張機能を読み込む"  
-> /blackGmail を指定
```
